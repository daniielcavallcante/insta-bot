Para executar o código, é necessária a instalação do framework Chrome Webdriver da Selenium (https://chromedriver.chromium.org/getting-started)
E também a iniciaização de um ambiente virtual no python (venv).

No caso acima, há um arquivo (accounts.txt) com diversos usuários. O programa percorre esse arquivo e comenta linha a linha na aba de comentários da publicação no instagram.
É possível personalizar todas as coisas. Conta a ser utilizada, lista de comentários e intervalo de tempo entre cada comentário (sleep).

É basicamente isto: um programa que comenta automaticamente em uma determinada publicação do instagram.
O instagram é realmente bom em detectar atividades anormais de contas. Por tanto, seja cuidadoso!
